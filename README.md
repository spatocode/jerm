# Jerm

Jerm is an open source tool that helps you deploy serverless app across all clouds.

## Quickstart

This guide will get you up and running in a few minutes.

### Install

To install Jerm run 

```
$ go install github.com/spatocode/jerm/cmd/jerm@latest
```

### Usage

Deploy your app

```
$ jerm deploy
```
